פעולות שוטפות — הכל בעמוד הניהול

העלאת שיחה חדשה

  1. היכנס לעמוד הניהול ← "＋ הוספת שיחה חדשה"
  2. בחר שנה, מלא כותרת ותיאור, בחר תאריך עברי (קובע את המיקום), מספר עמודים ומספר שיחות בקובץ
  3. תן שם קובץ באנגלית בלי רווחים (למשל vayeitzei-5707.pdf), צרף את ה-PDF ולחץ שמירה
  4. אם השנה עוד לא קיימת — קודם "＋ הוספת שנה"

עוד פעולות (בכפתורים ליד כל שיחה)

  • ✏️ עריכת כותרת/תיאור/תאריך/מונים
  • 🔄 החלפת ה-PDF בגרסה מתוקנת
  • 📄 שינוי שם הקובץ · 👁 הסתרה בלי מחיקה · 🗑️ מחיקה
  • ✏️ ליד שנה: שינוי שם, הערה, וסימון "התרגום הושלם"
  • בהגדרות למטה: קישור רשימת התפוצה, אימייל, מספר ביט

רשימת התפוצה והוואטסאפ

  • מייל: הנרשמים דרך טופס הגוגל נאספים אוטומטית. לצפייה בהם: פתח את הטופס ב-forms.google.com ← לשונית "תשובות". לשליחת עדכון — העתק את הכתובות ושלח מהג'ימייל (בשדה "עותק מוסתר"/BCC כדי לא לחשוף כתובות).
  • וואטסאפ: העדכונים נשלחים ע"י הודעה בקבוצה שיצרת. אם קישור ההזמנה דלף למי שלא צריך — בהגדרות הקבוצה אפשר לאפס את הקישור, ואז לעדכן את הקישור החדש בעמוד הניהול ← הגדרות.
  • שני הקישורים (טופס + קבוצה) מוגדרים בעמוד הניהול, באזור "הגדרות כלליות".

שליחת התראה (נוטיפיקציה) למנויים

גולשים שאישרו קבלת התראות באתר (או באפליקציה המותקנת) יקבלו הודעה לטלפון/מחשב בכל פעם שתשלח התראה. כך שולחים:

  1. היכנס ל-dashboard.onesignal.com (התחברות עם חשבון הגוגל שאיתו נרשמת)
  2. בתפריט למעלה: Messages ← New Message ← Push
  3. מלא כותרת (Title) והודעה (Message), למשל: "עלתה שיחה חדשה — ש״פ נח תש״ז"
  4. בשדה Launch URL הדבק קישור. לפתיחת האתר: https://sichotmeturgamot.github.io — או עדיף, קישור שפותח ישירות את השיחה החדשה לקריאה: https://sichotmeturgamot.github.io/read.html?file=pdfs/שם-הקובץ.pdf&title=שם השיחה (החלף את שם הקובץ ואת הכותרת)
  5. לחץ Review and Send ואשר. ההתראה מגיעה תוך שניות לכל המנויים
  • לצפייה בכמות המנויים: לשונית Audience
  • באייפון התראות עובדות רק אצל מי שהתקין את האתר כאפליקציה למסך הבית. באנדרואיד ובמחשב — גם מהדפדפן
  • מי שלחץ "לא עכשיו" יישאל שוב אוטומטית אחרי כמה ימים

אם משהו לא עובד

  • עדכנתי ולא רואה שינוי באתר: המתן כדקה ורענן. בטלפון — סגור ופתח את הדפדפן. במחשב — Ctrl+F5.
  • שגיאה 403 בעמוד הניהול: המפתח פג תוקף או חסר הרשאה. צור מפתח חדש: בקישור ניהול המפתחות למעלה ← Generate new token ← תוקף שנה ← Only select repositories ← בחר את המאגר ← Repository permissions ← Contents ← Read and write ← Generate. העתק והתחבר מחדש.
  • מחקתי משהו בטעות: הכל ניתן לשחזור דרך היסטוריית השינויים (קישור למעלה). אם צריך עזרה — כל קובץ שנמחק קיים שם בגרסאות קודמות.
  • נעזרים ב-Claude: אפשר תמיד לפתוח שיחה חדשה, לתת את כתובת האתר ולבקש עזרה — הכל בנוי מקבצים פשוטים שקל להבין.
